This tool collects domain names and IP addresses for specified targets by querying https://bgp.he.net/. It is highly recommended to manually check all the different company names your target is known as within https://bgp.he.net/ prior to running this tool against them.

# Help
```
➜ python bgp.py --help
usage: bgp.py [-h] [-d] [-v] targetCompanyNames [targetCompanyNames ...]

positional arguments:
  targetCompanyNames  target company names (e.g. "Google, Inc." "Google, LLC"
                      "Google Access LLC")

optional arguments:
  -h, --help          show this help message and exit
  -d, --debug         spawn Chrome browser in GUI mode
  -v, --verbose       print logging messages to STDOUT
```
